# VetAI Brand Guidelines

Interactive brand guidelines site for VetAI / VetAI — colors, typography, patterns, motion, layout, and illustrations in a responsive single-page experience.

## Stack

- [React 18](https://react.dev/) + [TypeScript](https://www.typescriptlang.org/)
- [Vite 6](https://vite.dev/)
- [Tailwind CSS 4](https://tailwindcss.com/)
- [Motion](https://motion.dev/) + [GSAP](https://gsap.com/) for animations

## Getting started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).

## Scripts

| Command             | Description                         |
| ------------------- | ----------------------------------- |
| `npm run dev`       | Start development server            |
| `npm run build`     | Type-check and build for production |
| `npm run preview`   | Preview production build locally    |
| `npm run lint`      | Run ESLint                          |
| `npm run typecheck` | Run TypeScript without emitting     |

## Project structure

```
src/
├── app/
│   ├── App.tsx              # Page composition
│   └── components/          # Sections and UI
├── assets/                  # Images and SVGs
├── hooks/                   # Shared hooks
├── imports/                 # Figma export assets
└── styles/                  # Global CSS and tokens
```
